Projetos

    API funcional com persistência em banco de dados relacional para cadastro e gerenciamento de pessoas, usando C#, Entity Framework e MySQL.

    Projeto de modelagem e criação de um banco de dados para cadastro de pessoas, com tabelas como pessoas, endereços, países e usuários, desenvolvido com MySQL Workbench e XAMPP.
